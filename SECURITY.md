# Security Policy

## Supported Versions

We only support the latest major versions of our projects.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < Last  | :x:                |

## Reporting a Vulnerability

We take security seriously at Nanoo Labs. If you find a vulnerability, please do NOT open a public issue.

Instead, please send an email to **hi@nanoolabs.dev**.

We will:

1. Acknowledge your report within 48 hours.
2. Work with you to understand and reproduce the issue.
3. Provide a timeline for a fix.
4. Give credit where it's due once the fix is public.

Stay safe.
